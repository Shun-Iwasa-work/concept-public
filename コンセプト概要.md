### 現在の方向性：「外にいるための宿」

**3/26の議論で #1〜#4 が一気に埋まった。**

```
「大人になって外にいる時間を失った人が、
 理由なく外にいられる場所。
 庭とデッキと焚き火とコーヒーがあって、
 部屋に戻る理由がない。
 グランピングは"外で遊ぶ"。うちは"外にいる"。
 何もしてないのに、帰ったら少し変わってる。」
```

| 軸 | 内容 |
|----|------|
| **動機** | 自分が外にいる暮らしを取り戻したい。人にも体験させたい |
| **誰のために** | 外にいる時間を失った大人（都市でデジタルの仕事をしている人） |
| **何を提供するか** | 「理由なく外にいられる」状態。プログラムではなく条件の設計 |
| **どうやって** | 広い庭・素足ウッドデッキ（屋根付き）・焚き火・コーヒー・ブランケット。外にいるハードルをゼロにする |
| **部屋の位置付け** | 寝床。主役ではない。外が主役 |
| **差別化** | グランピング＝外で遊ぶ。うち＝外にいる。それだけ |

### 「外にいる」が生む体験メカニズム（3/26議論で言語化）

```
引き算の空間（火・自然・静けさ・やることがない）
    ↓
脳が減速する
    ↓
普段しない会話が勝手に生まれる
    ↓
「何もしてないのに視点が変わった」
```

- リゾート → 楽しかった！（消費して満足）
- リトリート → 自分と向き合った（プログラムで変わる）
- **この宿 → 何もしてないのに、なぜかリセットされた（勝手に変わる）**

宿がやることは「普段しない会話が勝手に生まれる条件」を揃えるだけ。

| 条件 | 具体的に |
|------|---------|
| スマホを触る気にならない | 自然に囲まれて触る理由がない |
| やることがない | プログラムもアクティビティもない |
| 視線が合わない場所 | 焚き火を一緒に見る（向かい合うより楽） |
| 時間の区切りがない | 次の予定がない、チェックアウトまで自由 |
| 五感が自然に向く | 火の音、木の匂い、外の空気 |

### リノベ構想：「屋根のある外」（3/26議論）

河口湖古民家のような物件で、**玄関側の和室2部屋をくり抜いて巨大な縁側/テラスにする**構想。

| 要素 | 内容 |
|------|------|
| **屋根** | 既存の屋根をそのまま残す（雨でも外にいられる） |
| **床** | 和室の畳を撤去 → 木の板張り。素足で歩ける。寝転がれる。土間の開放感 + デッキの心地よさ |
| **外との境界** | 全面開放できる建具（折れ戸 or 引き込みサッシ）。開ければ壁がなくなる。閉めてもガラス越しに庭。「窓がない」感覚が理想 |
| **メイン空間** | この16畳超の「屋根のある外」が宿の主役。ここでコーヒーを飲み、本を読み、日向ぼっこし、寝転がる |
| **寝室等** | 最小限に整える。あくまで外で過ごすことが主眼 |

> **一言でまとめると：「屋根のある外」を作る。天気がいい日は全開で外。雨の日は閉めても外にいる気分。部屋じゃなくて、屋根付きの庭。**

### 候補A：犬と旅する人のための本気の拠点

**一行フック候補**：「犬と、旅をする」「犬が主役の1棟貸し」

| 観点 | 内容 |
|------|------|
| **なぜ作るか** | 犬と旅したくてもできなかった個人の体験。その不満を自分が解決する |
| **誰のために** | 犬と本気で旅したい人。妥協した旅ではなく、犬が主役の旅 |
| **何を解決するか** | 宿・食事・周辺スポット全部の「犬と入れるか問題」をまるごと解決 |
| **宿の設計** | 庭・洗い場・フェンス完備。キッチン充実で外食しなくても完結できる |
| **オーナーの役割** | 周辺の犬OKスポット（カフェ・飲食・トレイル）をキュレーションして案内 |
| **複数拠点への展開** | 各エリアで「そのエリアの犬連れ体験」をキュレーション → ブランドになる |
| **立地相性** | 山梨・長野（犬連れハイキング・川遊び）、湘南・三浦（海と犬） |

**強み**：「ペット可の宿」との差別化が明確。口コミが起きやすい。1棟貸しとの相性が抜群（他ゲストへの迷惑なし）。

**留意点**：インバウンドよりも国内ターゲット中心になりやすい。清掃コストが上がる可能性。

### 候補B：風土の宿 ―― 47都道府県、その土地が作った家

**一行フック候補**：「この家は、この土地が作った」「風土が設計した家に、泊まる」

| 観点 | 内容 |
|------|------|
| **コアコンセプト** | 各都道府県の気候・地形・四季・湿度に根差した建築技術を持つ職人が、地元材を使って作った宿。建物そのものが「その土地の環境への答え」 |
| **誰のために** | 「本物」を求めるインバウンド・国内のデザイン・建築好き。普通のホテルでは得られない体験を求める人 |
| **何を解決するか** | 「どこも同じ」になってしまった現代の宿泊施設。その土地でしか体験できない空間を作る |
| **宿の設計思想** | 地元の職人に「好きなように作らせる」。構造計算ではなく職人の目・経験・勘。曲がった木もそのまま使う。端材も家具になる。捨てるものがない |
| **スケール構想** | 投資家をつけて47都道府県展開。山梨の宿と湯河原の宿は根本的に違う顔になる。でも哲学は同じ |
| **コンセプトの強度** | 白川郷の合掌造り・沖縄の石積みと同じ論理。和辻哲郎「風土」の思想を体験できる場所。vernacular architectureとしてインバウンドにも刺さる |

**強み**：一棟ごとが唯一無二。展開するほどブランドが強くなる。職人文化の継承という社会的文脈もある。マクガフィンに取り上げられる哲学の強さがある。

**留意点**：フル新築・職人フルオーダーは初期コスト大。1棟目は古民家リノベ×地元職人で「風土の宿」の哲学を体現するところから始めるのが現実的。

**現在の位置付け**：中長期のビジョン・投資家向けストーリーとして記録。1棟目のコンセプト選定とは別に持っておく。

### 候補C：SOME YAMANASHI ―― 山梨の廃棄果実が、この色になった

**一行フック候補**：「山梨の土地が作った色に、泊まる」「捨てられるはずだったぶどうが、この布になった」

| 観点 | 内容 |
|------|------|
| **コアコンセプト** | 山梨の農家が廃棄する果実（ぶどうの皮・桃の葉・くるみの殻など）を染料として使い、宿の空間をその色で満たす。カッコよくないものをカッコよくする逆転の発想 |
| **誰のために** | デザイン・クラフト・サステナビリティに関心があるインバウンド。国内は30〜40代のクリエイティブ系グループ |
| **何を解決するか** | 「shiboriは知っているが体験できない」問題。「どこも同じ」宿泊施設問題。廃棄果実という社会課題への答え |
| **空間設計** | 宿のリネン・カーテン・暖簾・クッションがすべて藍染め（インディゴブルーを基調）。庭や縁側に染めた布が干してある |
| **体験設計** | ホストが事前に廃棄ぶどう皮等を煮出して染料液を作り置き。ゲストは「縛る→浸す→干す」だけで完結。講師不要のセルフ体験。季節ごとに使う素材・出る色が変わる |
| **素材の現実性** | ぶどうの皮（紫・グレー）は染色実証済み。冷凍保存で年間使用可能。山梨はぶどう生産量全国1位で廃棄品の調達が現実的 |
| **名前の強さ** | 「SOME」＝染め（日本語）＋some（英語で引っかかりになる）。「YAMANASHI」で場所を固定。SOME YUGAWARA等のシリーズ展開も可能 |

**強み**：「shibori」は海外ファッション・デザイン界で既に通じる言葉。廃棄物活用のストーリーは海外メディアが好きな文脈。庭に干された藍染めの写真は即シェアされる映え力がある。山梨でしか出せない色がある。

**留意点**：染料液の作り置き・管理がホストの定期作業になる。ぶどう皮染めの色は1年程度で退色するため、宿の備品（長期使用）には藍染め、体験（持ち帰り）には草木染めと役割を分ける運用が現実的。

**現在の位置付け**：1棟目のコンセプト筆頭候補。山梨物件との相性が特に高い（古民家・縁側・庭が確保しやすい）。

### 候補D：縁側と庭の宿 ―― 自分だけの日本庭園で、何もしない ★第一候補

**一行フック候補**：「自分だけの庭がある。あとは何もしなくていい。」「Do nothing, beautifully.」

| 観点 | 内容 |
|------|------|
| **コアコンセプト** | プライベートな小さな日本庭園を縁側から眺めながら、お茶と和菓子を楽しむ宿。「何もしない」を贅沢にする |
| **誰のために** | 「本物の日本」を静かに体験したいインバウンド。忙しい日常からリセットしたい国内カップル・小グループ |
| **何を解決するか** | 旅館の庭は「共有・遠くから眺めるもの」。1棟貸しには庭がない。プライベートで使える日本庭園が存在しない問題 |
| **空間設計** | 縁側から見える小さな日本庭園（苔・石・灯籠・飛び石）。縁側にはお茶と和菓子が常備。部屋から庭が見える間取り |
| **庭の作り方** | オーナーが石を選び、苔を植え、自ら作る。「この庭はオーナーが3ヶ月かけて作った」というストーリーがAirbnbリスティングの核になる |
| **体験設計** | 体験を用意しない、が体験。縁側に座る・庭を歩く・お茶を飲む・ぼーっとする。これだけで完結 |

**競合との差：**

| | 旅館 | 普通の1棟貸し | この宿 |
|---|---|---|---|
| 日本庭園 | ある（共有） | ない | **ある（プライベート）** |
| 縁側 | ある（共有） | ほぼない | **ある** |
| お茶・和菓子 | 部屋出し | ない | **常備・セルフ** |
| 他ゲストの目 | ある | ない | **ない** |

**強み**：「プライベートな日本庭園付き1棟貸し」は国内にほぼ存在しない。インバウンドに「縁側でお茶」は究極の日本体験として刺さる。庭の写真1枚でコンセプトが全部伝わる。オーナー自作のストーリーが差別化になる。「何もしない贅沢」は現代の旅行トレンドと合致する。

**留意点**：庭の初期造成コストが必要（小規模でも50〜200万円程度）。庭の維持管理（苔・剪定・清掃）が定期作業になる。物件選定時に「庭スペースが確保できるか」が必須条件。

**立地相性**：山梨◎（古民家・広めの土地が安価で確保しやすい）、湯河原○（温泉地の雰囲気と相性あり）

**現在の位置付け**：**★1棟目の第一候補コンセプト**。需要検証（Reddit r/JapanTravel への投稿）を経て正式採用する方針。物件内見時に「縁側と庭スペースが確保できるか」を最優先条件として確認する。

| 原体験 | ホテルは過剰か貧弱の二択。「ちょうどいい」がない |
| 体験の核 | 「アンチゴージャス、アンチチープ」— ちょうどいい、の心地よさ |
| 手段 | 無印の家具・タオル・アメニティで統一。装飾を排除。素材の質感で空間をつくる。価格も「ちょうどいい」帯 |
| ターゲット | 過剰な演出が苦手な30〜50代。「普通にいい部屋」がほしい人 |
| 差別化 | 高級ホテルの反対。ビジネスホテルの反対。「どちらでもない真ん中」 |
| フック | 「アンチゴージャス、アンチチープ」 |

### 岩佐の現在地（参考例との比較）

```
「大人になって外にいる時間を失った人が、
 理由なく外にいられる場所。
 庭とデッキと焚き火とコーヒーがあって、
 部屋に戻る理由がない。
 グランピングは"外で遊ぶ"。うちは"外にいる"。
 何もしてないのに、帰ったら少し変わってる。」
```

| 宿のタイプ | 主役 |
|-----------|------|
| ホテル・旅館 | 部屋（インテリア、ベッド、アメニティ） |
| グランピング | テント＋BBQ（アウトドア"体験"） |
| 星のや | 空間全体（引き算のラグジュアリー） |
| SANU | 家（自然の中の生活拠点） |
| **岩佐の宿** | **外（庭・デッキ・火・空）** |

残る論点：#5 ターゲット詳細（「外を失った大人」に国籍は関係ない。だが宿の作り方＝言語・集客・価格・空間の文脈が変わるため整合が必要）、#7 フック磨き。

## コンセプト設計フレームワーク

優れたコンセプトは以下の5要素が重なる部分に存在する。**ホテルと戦うのではなく、コンセプトを尖らせてマクガフィン（[参考](https://newstech.co.jp/service/mcguffin/)）に取り上げられるようなものを目指す。**

| 軸 | 問い | 現状の仮説 |
|----|------|------------|
| **① Target（誰に）** | 誰の、どんな不満を解決したいのか？ | 4人以上のグループ。普通のホテルでは「部屋を分けなければならない」不満を持つ人 |
| **② Value（何を）** | 顧客はそれを手にして、どう変化するのか？ | 1棟まるごと・1単位でまとまれる体験。ホテルでは出せないプライベート感と自由度 |
| **③ Method（どうやって）** | 他社には真似できない、どういった方法で届けるのか？ | 物件選び・内装・サービス設計でコンセプトを徹底する。価格を下げず体験に投資し続ける |
| **④ Positioning（どこの空白に）** | 競合と比べたとき、どの"空白地帯"にいるか？ | 「泊まれる場所」ではなく「○○な人のための体験のカテゴリ」を作る。近隣Airbnbと何が違うかを常に確認 |
| **⑤ Story / Hook（なぜ広まるか）** | 知らない人がシェアしたくなる"一行"があるか？ | Airbnbリスティング1行目・Instagram・メディア見出しになり得る文脈を持たせる |

> **物件内見との連動**：内見のたびに上記5軸を当てはめ、「この物件でどのコンセプトが活きるか」を記録していく。コンセプトは物件選びと相互にフィードバックしながら絞り込む。

## コンサル視点でのコメント（論点整理用）

以下は、小規模宿泊業の起業コンサルなら指摘しがちな論点。相談時のネタに。

### 立地の選び方

- **藤沢周辺（湘南・鎌倉・三浦）**  
  - メリット：近いので初期は現場確認・トラブル対応がしやすい。  
  - デメリット：家賃・土地代は高めのエリアが多く、競合も多い。  
- **山梨・長野・静岡など**  
  - メリット：家賃・土地が安いエリアを選べる。競合が少ないエリアも取りやすい。リモート・清掃外注前提なら「近さ」より「家賃・条件」を優先する選択肢になる。  
  - デメリット：遠いので、システム・委託でカバーする設計が前提。  

**整理**：「リモート＋清掃外注」を前提にするなら、**最初から「家賃が安く・競合が少ない田舎」でコンセプト勝負**に入る選択肢は十分ある。逆に「まず近場で実績を作ってから遠方に2棟目」という順番もあり。コンサルには「近場で始めるべきか、最初から田舎で始めるべきか」を、初期費用・時間・リスクの面から聞いておくとよい。

### 1年営業と民泊の関係

- **民泊（住宅宿泊事業法）**は、原則として**年間営業日数が180日まで**などの制限がある自治体が多い。  
- **1年を通して営業したい**なら、**簡易宿所**（旅館業法）などにすると制限なく営業できる場合が多い。物件・消防・建築の条件は変わるので、エリアと物件が決まったら「民泊でいくか簡易宿所でいくか」を許認可の観点で確認するのがよい。

### 田舎・安い物件でコンセプト勝負

- 「土地ではなくコンセプト・マーケで勝負」「競合と戦わない」「大人数・長期滞在・普通のホテルにないサービス」という方向性は、**立地のハンデを差別化で補う**戦略として一貫している。  
- コンサルには「長期滞在・大人数向けに、どんなサービス・訴求をすれば競合と被らないか」も相談すると、開業後のマーケ設計まで詰めやすい。
